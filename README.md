# Tkinter Learning Journey

A repository documenting my journey of learning **Tkinter**, Python's standard library for building desktop GUI (Graphical User Interface) applications.

This repository contains lesson-wise examples, experiments, and notes that build from the fundamentals to real GUI applications.

---

## 📚 Topics Covered

- Creating a Tkinter window
- Labels
- Styling Labels
- Buttons
- Updating Widgets with `config()`
- `pack()` Layout Manager
- `grid()` Layout Manager
- `rowspan` & `columnspan`
- `sticky`

### 🚧 Currently Learning

- Entry Widgets
- Frames
- Tkinter Variables
- Message Boxes
- Events

## 🎯 Goal

## 🎯 Goal

The goal of this repository is to build a strong understanding of **Tkinter** by learning concepts incrementally rather than simply following tutorials.

Each lesson introduces a new concept, reinforces previously learned topics through small examples and experiments, and builds toward larger applications.

The learning journey will culminate in building a fully functional desktop calculator using Tkinter, followed by more advanced desktop applications.

---

## 🛠 Requirements

- Python 3.14+
- Tkinter (included with the standard Python installation)
